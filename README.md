# MLtravel-Backend

NestJS API for MLtravel.

## Quick start

```bash
npm install
npm run start:dev    # http://localhost:3001
npm run gate:api     # typecheck + OpenAPI validate + Vitest
```

Health check: `GET http://localhost:3001/api/health`  
Swagger: `http://localhost:3001/api/docs?feature=LP-001-BE`
